# MQTT Push

Folgende Module beinhaltet das MQTT Push Repository:

- __MQTT Push Instances__ ([Dokumentation](MQTT%20Push%20Instances))  
	Überträgt zyklisch Gerätewerte als JSON-Nachrichten auf MQTT.

- __MQTT Push Variables__ ([Dokumentation](MQTT%20Push%20Variables))  
	Überträgt Variablenwerte bei Änderung und als Snapshot auf MQTT.

- __MQTT Push List__ ([Dokumentation](MQTT%20Push%20List))  
	Überträgt konfigurierbare Zählerwerte und Metadaten bei Wertänderung auf MQTT.
